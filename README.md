# Программа синхронизации локальной папки и папки на Яндекс.Диске

## Описание
Программа синхронизирует локальную папку с папкой на Яндекс.Диске.  
Основной источник данных — локальная папка, которая при запуске зеркально копируется в облако.

## Как работает
1. При первом запуске программа анализирует файлы в папке облака и выполняет зеркальную синхронизацию с учётом локального состояния.  
2. Используется бесконечный цикл с периодическим запуском:  
   - Если файл новый — загружается на Яндекс.Диск.  
   - Если файл изменён — перезаписывается на диске.  
   - Если файл удалён — удаляется из облака.

## Запуск
Запустить программу можно, выполнив скрипт `main.py`.

## Логгирование
Логи выводятся в консоль с уровнем DEBUG, где четко ясно
как работает код и что он делает.
Логи с уровнем INFO сохраняются в файле log_file.log.

## Конфигурация
В файле `.env`:
- `SYNC_FOLDER` — путь к локальной папке для синхронизации
- `CLOUD_FOLDER` — название папки на Яндекс.Диске
- `TOKEN` — OAuth токен для доступа к Яндекс.Диску
- `PERIOD_OF_SYNC` — интервал периодической синхронизации (в секундах)
- `LOG_FILE_PATH` — путь для логирования работы программы
